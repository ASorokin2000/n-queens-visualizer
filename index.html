How it works

The Minimum Conflicts algorithm:

  1. Start with random queen placement
  2. Pick a queen that's under attack (in red)
  3. Move it to the column with fewest conflicts
  4. Repeat until all queens are safe (in blue)

Tip: Try N=14 to see the algorithm in action!
